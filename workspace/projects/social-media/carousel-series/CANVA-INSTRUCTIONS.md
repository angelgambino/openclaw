# 💅 MADRIZZA — Canva Build Instructions
## 10-Minute Setup for All 3 Carousels

### Step 1: Open Canva
Go to canva.com → Create Design → Custom Size → **1080 x 1350 px**

### Step 2: Set Up Brand Kit
**Colors:**
- Background Dark: #1a1a2e
- Background Medium: #16213e  
- Background Accent: #0f3460
- Text White: #FFFFFF
- Text Muted: rgba(255,255,255,0.6) → use #9999AA
- Red (Before slide): #8B0000 background or #2D0000
- Green (After slide): #006400 background or #002D00
- Angel Club brand color for CTA slides (use your existing brand hex)

**Fonts:**
- Headlines: **Montserrat Extra Bold** (or your Angel Club brand font)
- Body: **Inter Regular** or **DM Sans**
- Stats/Numbers: **Montserrat Black** (oversized)

### Step 3: Create Master Template
1. Set background to #1a1a2e
2. Add footer bar at bottom: "angelclub.com/readytoraise" in #9999AA, 18pt, centered
3. Add Angel Club logo watermark bottom-left, 50% opacity
4. Save as template — duplicate for each slide

### Step 4: Build Each Slide

**For EVERY slide:**
- Duplicate the master template
- Headline: Montserrat Extra Bold, 72-82pt, white, centered
- Body text: Inter Regular, 28-32pt, white or muted
- Footer stays on every slide

**For Before/After slides:**
- BEFORE: Change background to dark red (#2D0000), add ❌ emoji
- AFTER: Change background to dark green (#002D00), add ✅ emoji
- Email text: Use italic or a monospace font to look like real email

**For CTA slides:**
- Add Angel's headshot (left-aligned or top)
- Larger font for "Ready to Raise"
- Include URL prominently
- Add Hubble/Pitch Slam URLs as secondary CTAs

### Canva Pro Tip
- Use "Brand Kit" to save colors + fonts
- Use "Bulk Create" to duplicate slides fast
- Export as PNG (not JPG) for crisp text on Instagram
- Export all slides as individual PNGs, then upload as carousel

### Alternative: Canva Template Search
Search Canva for "Instagram Carousel Dark Professional" — find one close to the aesthetic, then customize with your brand colors and the copy from the slide files.
