## 🎯 GLOBAL DESIGN

### Colors
| Name | Hex | Usage |
|------|-----|-------|
| Ivory | #F7F3EE | Primary background |
| Espresso | #3A2F2A | Primary text + primary buttons |
| Taupe | #EFE8DF | Alt section backgrounds |
| Muted Copper | #B8896B | Accent: secondary buttons, lines, icons |
| Soft Gold | #C9A96E | Premium accent: testimonial quotes, highlights |

### Typography
- Headlines: **Playfair Display** (64px hero → 42px section → 28px subsection)
- Body: **Inter** (18px desktop, 16px mobile)
- Accent text: **Inter Medium** (14px, uppercase, letter-spacing 2px — for social proof bar)

### Rules
- Mobile-first
- Max 3 colors per section
- High whitespace (minimum 80px between sections)
- No gradients, no shadows, no clutter
- All buttons minimum 48px tall (touch-friendly mobile)
- **Sticky mobile CTA** at bottom of screen: "Book Advisory Call"
